Welcome to Template’s documentation!¶

This repository serves as a template for python code. To understand what all these folders are here for see here. In some situations, you may not want all of this structure. In that case you can just delete folders you don’t want.

At a minimum, you should update the following aspects:

  • README.md: fill out

  • DemoPackage: rename to something representing your code e.g “FullySickAnalysis”

How to use Templates in github¶

  1. When creating a new repo, select a template from the dropdown list:

  1. OR, From a template in github, click ‘use this template’
